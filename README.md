# Uganda Poverty Classification System

Professional machine learning web application for predicting household poverty status in Uganda.

## Features

- User authentication (Individual, NGO, Government, Researcher)
- Single household prediction
- Bulk CSV upload for multiple households
- Prediction history with filters
- Analytics dashboard
- Downloadable reports (JSON/CSV)
- Dark/Light theme
- Admin user management

## Deployment on Streamlit Cloud

1. Upload all files to GitHub repository
2. Go to share.streamlit.io
3. Connect your GitHub repository
4. Select app.py as the main file
5. Click Deploy!

## Input Features (8)

1. Region
2. Region-Urban Type
3. Sub-Region
4. District
5. Urban/Rural
6. Monthly Income/Expenditure (UGX)
7. Household Size
8. Non-Food Expenses (UGX)

## Output Classes

- Poor
- Middle Class
- Rich

## Disclaimer

This application is for research and planning purposes only. Predictions are statistical estimates and should not be used for individual benefit determination.
